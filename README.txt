Description:
Fresh_tomatoes.html is a movie trailer website. The html for the website is generated using python.

Use: 
Open the fresh_tomatoes.html website in a web browser.

How it works:
entertainment_center.py is a python program that uses fresh_tomatoes.py and media.py to create fresh_tomatoes.html. 

media.py - is the file that contains the movie class

fresh_tomatoes.py - a file that takes a list of movies to create an html page (creates the actual fresh_tomatoes.html page)

entertainment_center.py - is the main program that creates a list of movies and calls a function to create the html for the movie trailer website
